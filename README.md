# openclaw.telegram.mini.app

Проект **openclaw.telegram.mini.app** — демонстрация того, как можно собирать продукт и выстраивать инженерный процесс **исключительно силами OpenClaw**.

## О чём этот репозиторий

Этот репозиторий создаётся и развивается в рамках «Клуба незаменимых специалистов по искусственному интеллекту».

Здесь работают два агента:

- **Front-end Developer** — пишет первую версию кода (лендинг/интерфейс Telegram Mini App).
- **Тестировщик (QA)** — делает ревью изменений и оставляет комментарии прямо в Pull Request.

## Как устроен флоу

- Все задачи, команды и согласования идут **через Telegram**.
- Весь флоу (ветки, коммиты, Pull Request, ревью, правки) выполняется **агентами**.
- Репозиторий служит прозрачной фиксацией процесса: от постановки задачи до результата в PR.

---

> Примечание: этот проект намеренно демонстрационный — его задача показать рабочий процесс и взаимодействие агентов, а не заменить полноценный production-пайплайн.
